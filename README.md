## 🚀 Projeto em destaque

Plataforma web para venda local de água, gás e produtos de limpeza.

O projeto possui:

* Catálogo de produtos

* Checkout via WhatsApp

* Controle de estoque
* Upload de imagens
* Permissões por cargo
